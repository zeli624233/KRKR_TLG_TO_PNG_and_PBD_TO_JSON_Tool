# 源代码参考说明

本工具参照了用户提供的两个源码/工具包：

## tlg2png-1.0

参考点：

- `AbstractTlgReader` 的 TLG0 / TLG5 / TLG6 自动识别逻辑。
- `Tlg0Reader` 的容器嵌套读取逻辑。
- `Tlg5Reader` 的 LZSS 解压、通道重建、差分恢复逻辑。
- `Tlg6Reader` 的 Golomb 解码、filter table、行恢复逻辑。
- 输出 PNG 的 RGBA 处理。

本工具中对应实现：

- `krkr_batch_tool/tlg_decoder.py`
- `krkr_batch_tool/tlg_convert.py`

## PBDConverter-main

参考点：

- `PBDConverter.exe -input="xxx.pbd"`：PBD 转 JSON。
- `PBDConverter.exe -input="xxx.json" -target="pbd"`：JSON 转 PBD。
- `PBDConverter.exe` 本质是 krkrz 的 `tvpwin32.exe` 改名。
- 运行目录需要 `PBDConverter.cf`、`data.xp3`、`json.dll`、`PackinOne.dll`。

本工具中对应实现：

- `krkr_batch_tool/pbd_convert.py`
- `pbd_converter_assets/PBDConverter.exe`
- `pbd_converter_assets/PBDConverter.cf`
- `pbd_converter_assets/data.xp3`

`json.dll` 和 `PackinOne.dll` 仍由用户从游戏 plugin 文件夹选择，软件自动复制到 `PBD文件解析配置`。
